# BIYAHE - Features

**Version:** 1.0
**Status:** Draft

# Purpose

This document defines all product features for Biyahe.

Features are prioritized according to business value and user impact.

Every feature must solve a real problem experienced by ride-hailing drivers.

# Out of Scope (MVP)

The following are intentionally excluded from Version 1.0.

- Per Trip Recording
- GPS Tracking
- Navigation
- Passenger Management
- Tax Computation
- Accounting Reports
- Payroll
- Fleet Dispatching

---

# Product Priorities

Priority 1

Help drivers finish the week successfully.

Priority 2

Remove manual Excel computations.

Priority 3

Improve financial visibility.

Priority 4

Support operator business rules.

Priority 5

Reduce financial stress.

---

# Product Success

Version 1.0 is successful if drivers can:

✅ Record daily operations in under one minute.

✅ Know their rental status instantly.

✅ Complete weekly closing without Excel.

✅ Know their true business profit.

✅ Understand where every peso went.
