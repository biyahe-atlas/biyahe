# 🚗 Biyahe

> Helping every driver finish the week financially healthy.

## Overview

Biyahe is a driver-first financial management platform designed for Filipino ride-hailing drivers and fleet operators.

Unlike traditional expense trackers, Biyahe helps drivers monitor weekly earnings, rental obligations, working capital, business expenses, cash reconciliation, and true take-home income.

Built from real-world driving experience.

## Vision

To become the financial operating system for every Filipino ride-hailing driver.
